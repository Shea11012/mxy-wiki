## UDP校验和
举例：
```
# 有以下数据
0110 0110 0110 0000
0101 0101 0101 0101
1000 1111 0000 1100

# 首先计算前两个之和
0110 0110 0110 0000
0101 0101 0101 0101
────────────────────
1011 1011 1011 0101

# 将得出的和于第三个字相加
1000 1111 0000 1100
1011 1011 1011 0101
────────────────────
0100 1010 1100 0001

# 将得出的和进行反码运算，这就是校验和
1011 0101 0011 1110
```
在接受方，将三个字与校验和相加，如果没有引入差错，则接受方的和将全是 1111 1111 1111 1111
