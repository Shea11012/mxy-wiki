```shell
echo -e "\e[1;31m颜色\e[0m"

# \e[1;31m 将内容变为红色
# \e[0m 还原设置
```

> \e = \033 = \x1b 这三种写法等同

**显示方式**：

| 数字| 意义 |
|:----:|-----|
|  0 | 终端默认设置|
| 1 | 高亮显示 |
| 2 |模糊 |
| 3 | 斜体 |
| 4 | 使用下划线 |
| 5 | 闪烁 |
| 6 | 闪烁 |
| 7 | 反白显示 |
| 8 | 不可见 |

常用 SGR 顔色
格式： 
```shell
\e[显示方式;前景色;背景色m
```

RGB 颜色
格式： 
```shell
38：表示前景色
48：表示后景色

\e[38;2;r;g;bm 内容 \e[0m
\e[48;2;r;g;bm 内容 \e[0m
```

![](https://mxy-imgs.oss-cn-hangzhou.aliyuncs.com/imgs/202108191845503.png)

256 色
格式：
```shell
\e[38;5;9m 内容 \e[0m
\e[48;5;14m 内容 \e[0m
```
