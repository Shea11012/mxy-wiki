### 组合状态

```mermaid
stateDiagram-v2
	[*] --> First
	state First {
		[*] --> A
		A --> [*]
	}
	
	state second {
		[*] --> B
		B --> [*]
	}
	
	state Third {
		[*] --> C
		C --> [*]
	}
	
	First --> second
	First --> Third
```

### 可选

```mermaid
stateDiagram-v2
	state if_state <<choice>>
	[*] --> IsPositive
	IsPositive --> if_state
	if_state --> False: if n < 0
	if_state --> True: if n > 0
```

### 分叉

```mermaid
stateDiagram-v2
	state fork_state <<fork>>
	[*] --> fork_state
	fork_state --> State2
	fork_state --> State3
	
	state join_state <<join>>
	State2 --> join_state
	State3 --> join_state
	join_state --> State4
	State4 --> [*]
```
